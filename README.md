# MLOps Homework 1: Воспроизводимый ML Pipeline

## Цель проекта
Создание воспроизводимого ML-пайплайна для классификации вина с использованием DVC для версионирования данных и MLflow для трекинга экспериментов.

## Структура проекта
\`\`\`

mlops_hw1/

├── data/              # Данные (версионируются через DVC)

├── src/               # Исходный код

├── dvc.yaml           # Описание пайплайна DVC

├── params.yaml        # Параметры подготовки и обучения

├── requirements.txt   # Зависимости

└── README.md         # Документация
\`\`\`

## Как запустить

1. **Установить зависимости:**
\`\`\`bash
pip3 install -r requirements.txt
pip3 install pyyaml joblib
\`\`\`

2. **Запустить пайплайн DVC:**
\`\`\`bash
dvc repro
\`\`\`

3. **Запустить MLflow UI для просмотра экспериментов:**
\`\`\`bash
mlflow ui --backend-store-uri sqlite:///mlflow.db
\`\`\`

4. **Просмотреть результаты:**
- Откройте http://localhost:5000 в браузере
- Модель и метрики будут доступны в интерфейсе MLflow

## Описание пайплайна

Пайплайн состоит из двух стадий:

1. **prepare** - подготовка данных:
   - Загрузка датасета Wine
   - Разделение на train/test
   - Сохранение обработанных данных

2. **train** - обучение модели:
   - Обучение RandomForest
   - Логирование в MLflow
   - Сохранение модели и графиков

## Воспроизводимость
Проект полностью воспроизводим через \`dvc repro\`
